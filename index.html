SOFTWARES

Unity

A powerful game development tool for both 2D and 3D. Cross-platform support, easy to learn and large community. Suitable for indie games to mid-range projects.

Unreal Engine

A high-end graphics engine, strong in 3D and cinematic games. Use Blueprint or C++ for programming. Optimized for high-quality games, realistic effects.

Autodesk Maya

Used for creating 3D models, rigging and animation. Popular in game and animation pipelines. Supports character creation, animation and simulation.

Adobe Substance Painter

Create detailed materials and textures for 3D models. PBR support - compatible with Unity and Unreal. Allows direct drawing on models and lighting testing.

Zbrush

Specializes in sculpting characters, high-detail monsters. Suitable for creating complex meshes, high-poly sculpt. Export model to game engine after retopology.

Autodesk 3DS Max

A powerful 3D modeling software, optimized for game design. Often used to build houses, environments, weapons. Supports UV unwrapping and exporting to game engines.

Adobe Illustrator

Create vectors, icons, interfaces and concept art for games. Suitable for drawing UI, logos or 2D characters. Light output, easy to integrate into Unity/Unreal.

Adobe Photoshop

Edit images, draw textures, design UI. Support drawing concept drawings, weapon skins, backgrounds. Essential tools for 2D and 3D game artists.

Adobe After Effect

Create motion effects, trailers and cinematics. Widely used in game introduction videos. Suitable for promotions, motion graphics and cutscenes.
